🔑 MESTER MÓD AKTÍV – helyes válaszok jelölve

1. Oldd meg a 2x + 3 ≤ 0 egyenlőtlenséget.

2x ≤ −3 ⇒ x ≤ −3/2 = −1.5.

2. Megoldás: −x + 5 ≤ 0.

−x ≤ −5 ⇒ x ≥ 5.

3. Oldd meg az 3x − 9 ≤ 0 egyenlőtlenséget.

3x ≤ 9 ⇒ x ≤ 3.

4. Megoldás: −4x + 8 ≤ 0.

−4x ≤ −8 ⇒ x ≥ 2.

5. Oldd meg: 5x + 0 ≤ 0.

5x ≤ 0 ⇒ x ≤ 0.

6. Megoldás: −2x + 6 ≤ 0.

−2x ≤ −6 ⇒ x ≥ 3.

7. Melyik a megoldása: 0·x + 5 ≤ 0?

0·x = 0 ⇒ 5 ≤ 0 soha nem teljesül.

8. Oldd meg: x ≤ 2x + 1.

x − 2x ≤ 1 ⇒ −x ≤ 1 ⇒ x ≥ −1.

9. Megoldás: 4x + 2 ≤ 2.

4x ≤ 0 ⇒ x ≤ 0.

10. Oldd meg: −3x + 0 ≤ 6.

−3x ≤ 6 ⇒ x ≥ −2.

11. Melyik a megoldása: 2x − 4 ≤ 2?

2x ≤ 6 ⇒ x ≤ 3.

12. Oldd meg: −x + 2 ≤ 4.

−x ≤ 2 ⇒ x ≥ −2.

13. Melyik x érték oldja meg: 0·x − 3 ≤ 0?

−3 ≤ 0 mindig igaz, bármely x.

14. Megoldás: 5x + 10 ≤ 10?

5x ≤ 0 ⇒ x ≤ 0.

15. Oldd meg: −6x + 12 ≤ 0.

−6x ≤ −12 ⇒ x ≥ 2.

16. Megoldás: 0·x + 0 ≤ 0?

0 ≤ 0 mindig igaz, bármely x.

17. Ha x ≤ −2 és x ≤ 3, mi igaz?

Mindkét feltétel teljesülése x ≤ min(−2,3)=−2.

18. Melyik x oldja meg: 2(x+1) ≤ 4?

2x+2 ≤ 4 ⇒ 2x ≤ 2 ⇒ x ≤ 1.

19. Oldd meg: −3(x−2) ≤ 6.

−3x+6 ≤ 6 ⇒ −3x ≤ 0 ⇒ x ≥ 0.

20. Megoldás: 4(x−1) ≤ 0?

4x−4 ≤ 0 ⇒ 4x ≤ 4 ⇒ x ≤ 1.

21. Ha ax + b ≤ 0, mi történik, ha a negatív?

Negatív együtthatónál fordul az irány: x ≥ −b/a.

22. Oldd meg: −2(x+3) ≤ 4?

−2x−6 ≤ 4 ⇒ −2x ≤ 10 ⇒ x ≥ −5.

23. Melyik megoldást írja helyesen: 6 − x ≤ 2?

6 − x ≤ 2 ⇒ −x ≤ −4 ⇒ x ≥ 4.

24. Az x + 5 ≤ 5 egyenlőtlenség megoldása:

x ≤ 0.

25. Ha a = 0, akkor ax + b ≤ 0:

0·x + b ≤ 0 ⇒ b ≤ 0, nincs x-től függő megszorítás.
