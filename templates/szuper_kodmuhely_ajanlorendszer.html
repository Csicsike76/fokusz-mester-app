1Az Adatbázis Létrehozása

Minden MI-projekt adatokkal kezdődik. Most létrehozzuk a saját, mini film-adatbázisunkat. Ez egy egyszerű "szótár" (dictionary), ahol a felhasználókhoz (Anna, Béla, stb.) hozzárendeljük az általuk látott és értékelt filmeket.

# 1. Lépés: Adatbázis
# A kulcs a felhasználó neve, az érték pedig egy újabb szótár,
# amiben a film címe és az értékelés (1-5) szerepel.
adatbazis = {
    'Anna': {'Star Wars': 5, 'Titanic': 1, 'Pókember': 4, 'Shrek': 5},
    'Béla': {'Star Wars': 5, 'Pókember': 5, 'Mátrix': 4},
    'Cecília': {'Shrek': 5, 'Jégkorszak': 4, 'Verdák': 3, 'Titanic': 2},
    'Dénes': {'Mátrix': 5, 'Star Wars': 4, 'Pókember': 5},
    'Eszter': {'Jégkorszak': 3, 'Verdák': 4, 'Titanic': 3},
    'Te': {'Shrek': 5, 'Star Wars': 4} # Ide kerül a mi ízlésünk!
}
Magyarázat: Létrehoztunk egy adatbazis nevű változót. A 'Te' kulcs alá írd be a saját kedvenc filmjeidet és értékeléseidet! Ez alapján fog a rendszer neked ajánlani.

2A Hasonlóság Kiszámítása

Hogyan találjuk meg azt a felhasználót, akinek a leginkább hasonló az ízlése a miénkhez? Ehhez egy matematikai módszert, az "euklideszi távolságot" fogjuk használni, de ne ijedj meg, a kód egyszerű!

# 2. Lépés: Hasonlóság Számítása
from math import sqrt

def hasonlosag(adatok, szemely1, szemely2):
    kozosen_ert_filmek = {}
    for film in adatok[szemely1]:
        if film in adatok[szemely2]:
            kozosen_ert_filmek[film] = 1

    if len(kozosen_ert_filmek) == 0:
        return 0

    negyzetosszeg = sum([pow(adatok[szemely1][film] - adatok[szemely2][film], 2)
                        for film in kozosen_ert_filmek])

    return 1 / (1 + sqrt(negyzetosszeg))
Magyarázat: Ez a hasonlosag nevű funkció két embert hasonlít össze. Megkeresi azokat a filmeket, amiket mindketten láttak, majd kiszámolja, mennyire térnek el az értékeléseik. A végeredmény egy 0 és 1 közötti szám, ahol az 1 a tökéletes egyezést jelenti.

3A "Lelki Társ" Megtalálása

Most írunk egy funkciót, ami végigmegy az összes felhasználón, és megtalálja azt, aki a leginkább hasonlít ránk.

# 3. Lépés: Legjobb Találat
def legjobb_talalatok(adatok, szemely, n=1):
    pontszamok = [(hasonlosag(adatok, szemely, masik), masik)
                  for masik in adatok if masik != szemely]
    pontszamok.sort(reverse=True)
    return pontszamok[0:n]
Magyarázat: A legjobb_talalatok funkció kiszámolja a hasonlóságot közted és mindenki más között, majd sorba rendezi őket. A végen visszaadja a legjobb találatot (a te "filmes lelki társadat").

4Az Ajánlások Generálása

Már majdnem készen vagyunk! Most írunk egy funkciót, ami megnézi, hogy a hozzánk legjobban hasonlító ember milyen filmeket látott, amiket mi még nem, és ezeket fogja nekünk ajánlani.

# 4. Lépés: Ajánlások
def ajanlasok(adatok, szemely):
    osszegek = {}
    hasonlosagi_osszegek = {}

    for masik in adatok:
        if masik == szemely:
            continue
        hasonlo = hasonlosag(adatok, szemely, masik)

        if hasonlo <= 0:
            continue

        for film in adatok[masik]:
            if film not in adatok[szemely]:
                osszegek.setdefault(film, 0)
                osszegek[film] += adatok[masik][film] * hasanlo
                hasonlosagi_osszegek.setdefault(film, 0)
                hasonlosagi_osszegek[film] += hasanlo

    ajanlott_filmek = [(osszeg / hasonlosagi_osszegek[film], film)
                      for film, osszeg in osszegek.items()]
    ajanlott_filmek.sort(reverse=True)
    return ajanlott_filmek

5Futtatás és Eredmény!

Most már csak annyi a dolgunk, hogy lefuttassuk a programot, és megnézzük, milyen filmeket ajánl nekünk!

# 5. Lépés: Futtatás
print("A te filmes 'lelki társad':")
print(legjobb_talalatok(adatbazis, 'Te'))

print("\nFilmek, amik tetszhetnek neked:")
print(ajanlasok(adatbazis, 'Te'))
