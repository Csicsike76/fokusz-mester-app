Egy ismeretlenes másodfokú egyenletek

🔑 MESTER MÓD AKTÍV – helyes válaszok jelölve

1. Melyik az általános másodfokú alak? ax² + bx + c = …

Általános alak: ax² + bx + c = 0.

2. Milyen diszkriminánst használunk? b² − 4ac = …

Diszkrimináns: Δ = b² − 4ac.

3. Ha Δ > 0, akkor a megoldások …?

Pozitív Δ esetén két különböző valós gyök.

4. Δ = 0 esetén a gyökök …?

Δ = 0 → dupla, egyetlen valós gyök.

5. Δ < 0 esetén a gyökök …?

Negatív Δ → komplex gyökpár.

6. Mely képlet adja a gyököket? x = …

Megoldóképlet: x = (−b ± √Δ)/(2a).

7. Oldd meg: x² − 5x + 6 = 0, gyökök?

Szorzatra bont: (x−2)(x−3)=0 → 2,3.

8. Oldd meg: x² + 4x + 4 = 0, gyök?

((x+2)² = 0) → x = −2 (dupla gyök).

9. Δ = b² − 4ac = 9 − 12 = −3, valós gyökök?

Negatív Δ esetén nincs valós megoldás.

10. Oldd meg: 2x² = 8 → x = …?

2x²−8=0 → x²=4 → ±2.

11. Oldd meg: x² + 1 = 0 → x = …?

x² = −1 → x = ±i.

12. Hol helyezkedik el a parabola csúcsa: x² − 4x + 3?

x₀=−b/2a=2, y₀=2²−8+3=−1.

13. Parabola tengelye: x² + 6x + 5?

x₀=−b/2a=−6/2=−3.

14. Hol metszi az y-t: x² − 2x − 3?

y-nél: x=0 → y=−3.

15. Oldd meg formula nélkül: x² − 9 = 0?

x²=9 → ±3.

16. Mikor van parabola teteje negatív?

a<0 esetén nyitott lefelé.

17. Oldd meg: 3x² + 6x + 2 = 0?

Δ=36−24=12, x= (−6±√12)/6=−1±√3/3.

18. Parabola nyitása: x² + bx + c az a=1?

a=1 → alapértelmezett nyitás felfelé.

19. Mely két gyök közt lesz a parabola negatív?

a>0 esetén negatív ahol érték alatt van x tengely.

20. Melyik oldás ismeretlen nélküli: x²+bx+c=0?

Megoldóképlet mindig működik.

21. Mi a csúcs koordinátája általánosan?

x₀=−b/2a, y₀=−Δ/4a.

22. Δ kiszámítása: ax²+bx+c, ha a=2,b=3,c=1?

Δ=9−8=1.

23. Hol metszi a parabola x tengelyt: x²−4=0?

x²=4 → x=±2.

24. Melyik jellemző: parabola egyenesre képezése?

Derivált zérus helye a csúcs.

25. Hogyan nevezzük: ax²+bx+c alakú függvényt?

Másodfokú függvény grafikonja parabola.
