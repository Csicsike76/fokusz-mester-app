1. Valós számok halmaza

1.1 Természetes számok négyzetének négyzetgyöke

Egy természetes szám négyzetének négyzetgyöke visszaadja az eredeti számot.

√(n²) = n, ahol n ∈ ℕ
Példa: √49 = 7, mert 7² = 49.

1.2 Irracionális számok

Olyan valós számok, amelyek nem írhatók fel két egész szám hányadosaként. Tizedes alakjuk végtelen és nem szakaszos.

√2 ≈ 1,41421356…
Példa: √2, π, √3 mind irracionálisak.

√50 = √(25·2) = 5√2
Példa: √72 = √(36·2) = 6√2

1.4 Valós számok ábrázolása, összehasonlítás, abszolút érték

Minden valós szám egyértelműen ábrázolható a számtengelyen. Az abszolút érték a számtengelyen mért távolság 0-tól.

|x| = x, ha x ≥ 0; |x| = –x, ha x < 0
Példa: |–5| = 5; |3,2| = 3,2

1.5 Műveletek valós számokkal

Az összeadás, kivonás, szorzás, osztás ugyanúgy működik irracionális számokkal is, mint racionálisokkal.

a√b ± c√b = (a ± c)√b
Példa: 3√5 + 2√5 = 5√5

Két pozitív szám számtani közepe: (a+b)/2; mértani közepe: √(ab)

ma = (a + b)/2  mg = √(ab)

x² = 25 ⇒ x = 5 vagy x = –5

2. Lineáris egyenletek és egyenletrendszerek

2.1 Ekvivalens egyenlőségek és azonosságok

Két egyenlet ekvivalens, ha ugyanazt a megoldáshalmazt adják.

ax + b = 0 ⇔ ax = –b
Példa: 2x – 4 = 0 ⇔ 2x = 4 ⇔ x = 2
Gyakorló Kvíz a Témakörben →

2.2 ax + b = 0 alakú egyenletek

Elsőfokú egyismeretlenes egyenlet megoldása:

x = –b/a (a ≠ 0)
Példa: 3x + 6 = 0 → 3x = –6 → x = –2

2.3 Kétismeretlenes lineáris egyenletrendszerek

Két lineáris egyenletből álló rendszer megoldása helyettesítéssel vagy kiküszöböléssel.

2x + 3y = 12
x – y = 1
Példa megoldás: x = 3, y = 2

3. Az adatszervezés elemei

3.1 Két halmaz Descartes-szorzata

Az A és B halmaz Descartes-szorzata az összes lehetséges (a,b) rendezett párok halmaza, ahol a∈A és b∈B.

A × B = { (a,b) | a ∈ A, b ∈ B }
Példa: Ha A = {1,2}, B = {x,y}, akkor A × B = {(1,x),(1,y),(2,x),(2,y)}.
Gyakorló Kvíz →

3.2 Derékszögű koordináta-rendszer

Két egymásra merőleges számtengely alkotja: vízszintes x, függőleges y tengely. Minden pontot (x;y) koordináták írnak le.

P(x;y)
Példa: A(3;–2) pont 3 egységre van az origótól jobbra és 2 egységre lefelé.
