1. Oldd meg: x² − 7x + 10 = 0

Oldjuk meg a következő másodfokú egyenletet:
x² − 7x + 10 = 0

🔹 1. Általános alak
Ez egy másodfokú egyenlet: ax² + bx + c = 0
ahol: a = 1, b = −7, c = 10

🔹 2. Szorzatra bontás módszer
Keresünk két számot, amelyek:
• összege = 7 (mivel b = −7)
• szorzata = 10 (mivel c = 10)

🔹 3. A számok megtalálása
2 + 5 = 7 ✓
2 × 5 = 10 ✓

🔹 4. Faktorizálás
x² − 7x + 10 = (x − 2)(x − 5) = 0

✅ Megoldás:
x − 2 = 0 → x = 2
x − 5 = 0 → x = 5

Ellenőrzés:
x = 2: 4 − 14 + 10 = 0 ✓
x = 5: 25 − 35 + 10 = 0 ✓

2. Megoldás: 2x² + 5x + 2 = 0

Oldjuk meg a következő másodfokú egyenletet:
2x² + 5x + 2 = 0

🔹 1. Általános alak
ahol: a = 2, b = 5, c = 2

🔹 2. Megoldóképlet alkalmazása
x = (−b ± √(b² − 4ac)) / (2a)

🔹 3. Diszkrimináns számítása
D = b² − 4ac = 5² − 4×2×2 = 25 − 16 = 9

🔹 4. Gyökök számítása
x = (−5 ± √9) / (2×2) = (−5 ± 3) / 4

x₁ = (−5 + 3) / 4 = −2/4 = −0.5
x₂ = (−5 − 3) / 4 = −8/4 = −2

✅ Megoldás: {−2, −0.5}

Faktorizált alak:
2x² + 5x + 2 = (2x + 1)(x + 2) = 0

3. Oldd meg Δ módszerrel: x² + 4x + 1 = 0

🔹 2. Diszkrimináns (Δ) számítása

🔹 3. Δ értékelése
Mivel Δ = 12 > 0, az egyenletnek két különböző valós gyöke van.

🔹 4. Megoldóképlet
x = (−b ± √Δ) / (2a)
x = (−4 ± √12) / (2×1)
x = (−4 ± √(4×3)) / 2
x = (−4 ± 2√3) / 2

✅ Megoldás:
x₁ = −2 + √3 ≈ −0.27
x₂ = −2 − √3 ≈ −3.73

6. Hol van a csúcs: x² − 6x + 5?

7. Milyen típusú megoldás, ha Δ = 0?

🔹 Diszkrimináns (Δ) definíciója

🔹 Ha Δ = 0:
• Az egyenletnek pontosan egy valós gyöke van
• Ez a gyök "dupla gyök" vagy "kétszeres gyök"
• A parabola érinti az x-tengelyt (nem metszi)

🔹 Példa:

x = −4/2 = −2 (dupla gyök)
Faktorizálva: (x + 2)² = 0

✅ Válasz: Dupla gyök

🔹 2. Ha a > 0:

🔹 3. Ha a < 0:

A legmagasabb kitevő szerint nevezzük el:
• 2. fokú → másodfokú egyenlet
• 1. fokú → elsőfokú (lineáris) egyenlet
• 3. fokú → harmadfokú egyenlet

✅ Válasz: Másodfokú egyenlet

🔹 3. Feltételek
Ahhoz, hogy valóban másodfokú legyen:
• a ≠ 0 (különben nem lenne x² tag)
• b és c lehet 0 is

🔹 4. Egyenlet típusok
• Teljes másodfokú: ax² + bx + c = 0 (a,b,c ≠ 0)
• Hiányos másodfokú: ax² + c = 0 (b = 0)
• Hiányos másodfokú: ax² + bx = 0 (c = 0)

🔹 5. Példák
• x² + 5x + 6 = 0 → másodfokú
• 3x + 7 = 0 → lineáris (elsőfokú)
• x³ + 2x − 1 = 0 → harmadfokú
• 2x² − 8 = 0 → másodfokú (hiányos)
